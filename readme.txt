A simple and nice janeman theme core plugin for elementor Widget.

== Description ==
Introducing the Janeman Theme Core Plugin for Elementor! This simple yet elegant widget empowers you to effortlessly add sections with drag-and-drop functionality. Customize your sections with ease and elevate your design, all within Elementor.

For Free & Premium WordPress Theme visit : [AnnurTheme](https://annurtheme.com/).

== Installation ==
Installation is fairly straight forward. Install it from the WordPress plugin repository.
